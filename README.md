# KaashaAriv — AI Financial Intelligence Workspace

KaashaAriv is a full-stack AI-powered financial intelligence platform that transforms corporate annual reports into structured executive insights using Generative AI.

---

## 🚀 Features

- 📄 Upload annual financial reports (PDF)
- 🧠 AI-generated executive summaries using Gemini
- 📊 Extracts key financial insights and risks
- 📈 Executive-grade dashboard UI
- ⚡ Real-time backend processing
- 🧩 Modular full-stack architecture

---

## 🏗️ Tech Stack

### Frontend
- Next.js (App Router)
- TypeScript
- Tailwind CSS
- shadcn/ui
- Axios

### Backend
- FastAPI
- Python
- Google Gemini API
- Uvicorn

---


## ⚙️ How It Works

1. Upload PDF report
2. Backend extracts document content
3. Gemini AI processes and analyzes data
4. Structured financial insights are generated
5. Frontend renders an executive-grade dashboard

---

## 🧠 AI Capabilities

- Executive Summary Generation
- Business Insight Extraction
- Risk Identification
- Financial Interpretation

---

## 🖥️ Local Setup

### Backend Setup

cd backend  
pip install -r requirements.txt  
uvicorn main:app --reload  

---

### Frontend Setup

cd frontend  
npm install  
npm run dev
